# BuyOrWait – AI-Based Food Price Forecasting System

BuyOrWait is a Green AI–driven decision support system that predicts short-term price trends of essential food commodities such as vegetables and pulses. It helps consumers decide whether to buy now or wait for a better price, reducing unnecessary spending and food waste.

## Features
- Predicts future prices using Machine Learning
- Recommends best day to buy
- Displays lowest expected price and date
- Interactive AI assistant for user queries
- Visual price trend analysis

## Technology Stack
- Frontend: Streamlit
- Backend: Python
- Data Processing: Pandas, NumPy
- Visualization: Matplotlib
- Machine Learning: Scikit-learn (Linear Regression)
